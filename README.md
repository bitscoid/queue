# SvelteKit Starter Kit

Starter kit dengan:

- ✅ SvelteKit + TypeScript
- ✅ TailwindCSS + DaisyUI
- ✅ Prisma + PostgreSQL
- ✅ Auth via JWT + Cookie
- ✅ Layout Navbar Responsive
- ✅ Login page siap pakai

## Cara Mulai

1. Clone project ini:
   ```bash
   npx degit rizaputrait/sveltekit-starter my-app
   cd my-app
   npm install
   ```
